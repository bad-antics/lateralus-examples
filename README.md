
## Contributing ❤️

**We'd love your examples!** See [CONTRIBUTING.md](CONTRIBUTING.md) for guidelines.

The short version:
1. Fork this repo
2. Add your `.ltl` file in the appropriate folder
3. Include a comment header explaining what it does
4. Open a PR

Every contribution adds `.ltl` files to GitHub under your account — helping the entire Lateralus ecosystem grow.
